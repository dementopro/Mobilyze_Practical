# Mobilyze Google Maps Task

This project is a React application that allows users to add, remove, export, and import locations on a Google Map.

## Getting Started

To get started with this project, follow these steps:

1. Clone this repository to your local machine.
2. Install dependencies using `npm install`.
3. Obtain a Google Maps API key from the Google Cloud Console.
4. Create a `.env` file in the project root and add your API key like so:
5. Run the application using `npm start`.
6. Open your browser and navigate to `http://localhost:3000` to view the application.

## Usage

- Click on the map to add a new location marker.
- Click on an existing marker to view its coordinates and remove it.
- Export the current list of locations to a CSV file.
- Import a list of locations from a CSV file.

## Developer API Key

Please note that this project uses a developer API key for Google Maps. Make sure to add the api key
 <LoadScript googleMapsApiKey=""> with your own API key obtained from the Google Cloud Console.
